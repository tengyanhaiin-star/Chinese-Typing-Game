# 中文打字训练

一个无需安装、单文件运行的中文打字练习小游戏。打开 `typing-trainer.html` 即可开始，支持电脑和手机浏览器。

## 这是什么

游戏围绕"看拼音、打汉字"展开，词库包含800个常用中文词组，每个词组都标注了拼音。游戏分为两种模式：一种是传统的限时打字测试，另一种是把打字练习包装成一场3D生存战斗。

## 限时计分模式

最朴素的打字练习：60秒内尽可能多地正确输入屏幕上显示的词组。每答对一个，词库会立刻换下一个，连续答对则越打越快。模式结束后会显示你的总得分。

适合用来热身，或者单纯检验自己对常用词汇的反应速度。

## 无尽模式

这是游戏的核心玩法，把打字练习变成了一场3D空间防御战。

**基本规则**：词组组成的几何体敌人会从四面八方向屏幕中心（也就是你的角色所在的位置）移动，你需要尽快输入正确的词组来消灭它们。如果敌人撞到中心，你会扣血；血量归零，游戏结束。

**敌人形状与伤害**：敌人一共有六种几何形态，移动速度越慢的敌人造成的伤害越高：

| 形状 | 抵达中心所需时间 | 造成伤害 |
|------|------|------|
| 正四面体 | 9-10秒 | 5点 |
| 正方体 | 11-12秒 | 6点 |
| 正八面体 | 13秒 | 7点 |
| 正十二面体 | 14秒 | 8点 |
| 正二十面体 | 15秒 | 9点 |
| 球体 | 16秒 | 10点 |

**大招系统**：每正确输入20个字符，会随机获得一个大招，使用后清空充能重新计数。大招有六种，覆盖防御、控制、回复和清场等不同策略：

| 大招 | 概率 | 效果 |
|------|------|------|
| 护盾 | 30% | 获得一层护盾（最多叠加3层），可抵消一次伤害 |
| 冻结 | 40% | 冻结场上所有敌人3秒，期间敌人停止移动；新生成的敌人不受影响 |
| 复活甲 / 医疗包 | 10% | 没有复活甲时获得复活甲（血量归零时自动触发，恢复至20点血量，仅能使用一次）；已有复活甲时则改为获得医疗包（直接回复20点血量） |
| 物换星移 | 10% | 将场上所有敌人传送到新的随机位置，重新计算抵达中心的距离 |
| 命运审判 | 10% | 必定消灭离中心最近的敌人，其余每个敌人有40%概率被一并消灭；场上没有敌人时大招浪费 |

冻结的概率最高，是最常使用的控制手段；命运审判则是应对场面失控时的保底清场手段。

## 操作方式

点击文字框，直接打字即可，输入框会自动聚焦。无尽模式下按 `Esc` 键可以随时退出回到主界面。

为了防止投机取巧，游戏禁用了粘贴功能，也无法通过框选复制题目文字。

## 技术实现

整个游戏是一个不依赖任何构建工具的单文件 HTML，把 HTML、CSS 和 JavaScript 全部写在一起，双击即可在浏览器中打开运行。

- **3D 渲染**：无尽模式的场景使用 [Three.js](https://threejs.org/)（r128）搭建，敌人模型使用 Three.js 内置的多面体几何体（四面体、立方体、八面体、十二面体、二十面体、球体）
- **音效**：所有音效通过浏览器原生的 Web Audio API 实时合成，没有使用任何外部音频文件
- **字体**：界面正文使用 Google Fonts 的 Noto Sans SC（字重400），拼音部分单独使用 Inter 字体，以获得更清晰的字母显示效果
- **兼容性**：针对 iOS 触屏设备做了专门的事件处理优化，避免触摸操作导致的交互失灵问题。注意：不兼容手机，请勿在手机上玩此游戏。
